# Формат пака
Скиллы
Каждый скилл с присавкой:

`{название_пака}:{название}-{название}/SKILL.md`
(заполни плейсхолдеры)





**Оптимизация:**

пак должен иметь возможность расходовать токенов как можно меньше. разработке пока нужно предложить различные способы оптимизаций

Например:

Каждый markdown должен ссылаться на главный README. Тогда когда нужно что-то изменить везде — агент идет по этим ссылкам, как по карте. Быстрее чем лазить по папкам.

И даже если инструкции имеет определенный отличительный формат от других файлов — смысл один: пройтись по всем markdown-ам через их связи, найти нужное и поменять.
