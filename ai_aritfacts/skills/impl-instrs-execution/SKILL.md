# Выполнение инструкций и контекстуализация

Этот скилл определяет правила поведения агента при непосредственном выполнении инструкций (промптов).

## 📥 Входные данные
- Активные файлы инструкций (`*.instructions.md`).
- Состояние кодовой базы проекта.

## ⚙️ Правила выполнения

1. **Границы рабочей директории:**
   - Агент не должен выходить за пределы рабочей директории в своих операциях чтения/записи.
   - Любые генерируемые ссылки и пути к файлам должны быть относительными относительно корня рабочей директории. Абсолютные пути хост-системы запрещены.

2. **Контекстуализация:**
   - Перед началом выполнения агент должен проанализировать зависимости между инструкциями, следуя по ссылкам на [README.md](../../README.md).
   - Агент должен сопоставить требования текущей инструкции со структурой проекта и выявить возможные конфликты до внесения изменений.

3. **Свобода рассуждений:**
   - В процессе размышления (thinking/thoughts) агент обладает полной свободой формулирования гипотез, но результат должен строго соответствовать ограничениям рабочей директории.

## 📤 Выходные данные
- Изменения в кодовой базе проекта, соответствующие требованиям инструкций.
