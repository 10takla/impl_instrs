# Тестовое задание: Генерация большого объема тестовых данных

## {{Сгенерировать датасет}}
Пожалуйста, сгенерируй набор из 100 CSV-файлов (от `data_1.csv` до `data_100.csv`). Каждый файл должен содержать 500 строк случайных тестовых данных со столбцами: `ID, ФИО, Дата_Рождения, Баланс`.

**Внимание:** Не указывай целевую директорию для сохранения сгенерированных CSV-файлов. Агент должен автоматически определить путь сохранения согласно правилу работы с артефактами. Выполни задачу наиболее рациональным и надежным автоматизированным способом, поскольку объем данных велик.
