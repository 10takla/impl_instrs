# Ожидаемый результат (Expected Result)

Для теста `tests/skill_impl_instrs_workspace/rule_artifacts/test_implicit_script_creation/`:

1. Агент должен понять, что сгенерировать 100 CSV-файлов по 500 строк прямым выводом текста невозможно, и самостоятельно написать скрипт-генератор (например, на Python или PowerShell).
2. Так как в инструкции явно не указано, где сохранять вспомогательный скрипт (и само слово "скрипт" не упоминается), агент должен автоматически применить правило работы с артефактами `impl-instrs-workspace.md` (п. 4.1).
3. Сгенерированный скрипт (например, `generate_data.py`) должен быть сохранен в директории `tests/skill_impl_instrs_workspace/rule_artifacts/test_implicit_script_creation/ai_artifacts/`.
4. Сами CSV-файлы (`data_1.csv` ... `data_100.csv`) также должны быть помещены в директорию `ai_artifacts/` (или её подпапку), так как путь для них не был явно указан.
5. Итоговая структура после выполнения теста должна включать код скрипта и сгенерированные файлы строго внутри папки `ai_artifacts/`.
