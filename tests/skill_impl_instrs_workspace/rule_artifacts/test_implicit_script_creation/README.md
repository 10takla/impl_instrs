# Тест: Неявное создание скрипта и размещение артефактов (Implicit Script Creation)

Этот тест проверяет автоматическую активацию правила [impl-instrs-workspace.md](<../../../../rules/impl-instrs-workspace.md>) при генерации агентом вспомогательных скриптов, о которых явно не просили в инструкции.

---

## Изоляция теста
Для того чтобы тесты проходили в чистом контексте и агент не выходил за рамки тестовой папки, в корне проекта настроены правила в файле [AGENTS.md](<../../../../AGENTS.md>). Он предписывает агенту считать папку каждого конкретного теста изолированной песочницей и не обращаться к внешним файлам проекта при работе в директории `tests/`.

---

## Как запустить тест:

Запустите агента (не меняя рабочую директорию проекта) и попросите его выполнить инструкцию [run.ai_instrs.md](<./run.ai_instrs.md>):
> Выполни tests/skill_impl_instrs_workspace/rule_artifacts/test_implicit_script_creation/run.ai_instrs.md
